## Prerequisites

Before starting these tutorials, you should have:

- **Python 3.11 or later** installed on your system
- **Nauyaca installed** - See the [Installation Guide](../installation.md) if you haven't already
- **Basic Python knowledge** - Understanding of functions, async/await, and basic command-line usage
- **Text editor** - Any editor you're comfortable with

## Recommended Learning Path

If you're new to Nauyaca or the Gemini protocol, we recommend following the tutorials in this order:

1. **Start with [Your First Capsule](your-first-capsule.md)** to understand the server side and Gemini content format
2. **Move to [Building a Client](building-a-client.md)** to see how clients interact with capsules
3. **Finish with [Securing Your Server](securing-your-server.md)** to learn production-ready security practices

Each tutorial is self-contained, so you can also jump directly to the one that interests you most.
