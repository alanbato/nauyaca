## How to Use Reference Documentation

Reference documentation is organized for **lookup**, not **learning**:

- **Looking for a specific command?** Check the [CLI Reference](cli.md)
- **Need to configure a feature?** See the [Configuration Reference](configuration.md)
- **Wondering what a status code means?** Consult the [Status Codes](status-codes.md)
- **Integrating Nauyaca in Python?** Browse the [API Reference](api/index.md)

!!! tip "New to Nauyaca?"
    If you're learning Nauyaca for the first time, the reference documentation may be overwhelming. Start with the [Getting Started tutorial](../quickstart.md) instead, which walks you through the basics step by step.
